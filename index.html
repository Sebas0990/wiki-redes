Final thoughts

To start with the creation of the simple network, we used a wireless router, the “internet” and a cable modem. After that we linked a Personal Computer, a smartphone and 2 laptops to the wireless modem, and the main server to the “internet”.

We connected the Personal Computer through a “Copper Straight-Through” (UTP-45) cable to the wireless router, afterwards, both laptops had to be turned off in order to change the main connectivity module from a cable one to a wireless one (because it apparently is 2007) and restarting the laptops, now then, on the “desktop” window of the devices, we access the application called “PC Wireless”, and from there, access to “connections”, we await for the name of the network to appear, and as soon as it appears on screen, we choose said option in order to connect.

In order to connect the smartphone, we access the configuration screen, and have to manually input the name (SSID) of the network (again, 2007), and since this network has no access verification (password), it will connect automatically without any more steps.

Connections

In this simple but thoughtful experiment, we were able to learn how to manage a simple network which connects in a simple array to the internet, which in turn itself connects to a server, all of this while using both cabled and wireless connectivity, which connected either automatically or manually in the case of the smartphone.
